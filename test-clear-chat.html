🧪 Clear Chat Functionality Test

📋 Test Instructions

This page tests the new "Clear Chat" functionality. Follow the steps below to verify everything works correctly.

🔍 Test Steps

  1. Open the chat widget by clicking the floating chat button
  2. Send a few messages to the AI assistant (e.g., "Hello", "Tell me about shoes")
  3. Verify products appear in the right sidebar
  4. Click the "New Chat" button (🗑️ New Chat) at the bottom of the chat
  5. Verify the following:
    • All messages are cleared except the initial "Hello! I'm your AI assistant..." message
    • Product sidebar is empty
    • New chat-id and hash-key are generated (check browser console)
    • Input field is focused and ready for new messages
  6. Send a new message to verify the chat works normally after clearing
💡 Expected Behavior: The clear chat function should reset the conversation while preserving the initial hello message, generate new session identifiers, and clear all product recommendations.
✅ Success Criteria: After clicking "New Chat", you should see only the hello message, empty product sidebar, and be able to start a fresh conversation with new session identifiers.
