Online Assistant Widget Demo

This page demonstrates the Assistant Widget functionality. Look for the floating chat button in the bottom-right corner!

Features:

  • Floating chat button with smooth animations
  • Responsive chat window that adapts to screen size
  • AI-powered responses and product suggestions
  • Customizable styling to match your website
  • Easy integration with any website

Integration Instructions:

To integrate this widget into your website, simply add the following code:

<div id="assistant-widget"></div>
<script src="./dist/assistant-widget.umd.js"></script>
<script>
  AssistantWidget.createWidget('#assistant-widget', {
    apiUrl: 'https://your-cms-api.com/chat',
    title: 'Online Assistant',
    primaryColor: '#007bff'
  });
</script>
